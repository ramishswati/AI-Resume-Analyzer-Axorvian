# 📄 ResumeAI — AI Resume Analyzer

An intelligent AI-powered Resume Analyzer developed using **Google Gemini API** and **Streamlit** as part of the **Axorvian Internship Program — Task 3**.

![Python](https://img.shields.io/badge/Python-3.10+-blue?style=flat-square&logo=python)
![Streamlit](https://img.shields.io/badge/Streamlit-Latest-red?style=flat-square&logo=streamlit)
![Gemini](https://img.shields.io/badge/Gemini-2.5%20Flash-purple?style=flat-square&logo=google)

## ✨ Features

- 📤 PDF Resume upload and text extraction
- 📋 Job Description input
- 🎯 AI-powered Resume Match Score (0-100%)
- ✅ Matched Skills detection
- ❌ Missing Skills identification
- ⭐ Additional Skills display
- 💪 Strengths & Weaknesses analysis
- 💡 AI-powered improvement suggestions
- 👤 Candidate information extraction
- 📊 Visual progress bar

## 🛠️ Tech Stack

- **Python** — Core language
- **Google Gemini API** — AI/NLP analysis engine
- **Streamlit** — Web interface
- **pdfplumber** — PDF text extraction
- **python-dotenv** — Environment management

## 🚀 Installation

```bash
git clone https://github.com/ramishswati/AI-Resume-Analyzer-Axorvian.git
cd AI-Resume-Analyzer-Axorvian
pip install -r requirements.txt
streamlit run app.py
```
Or paste your API key directly in the sidebar.
